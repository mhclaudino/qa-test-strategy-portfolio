# Work-in-Progress Evidence

This directory contains controlled evidence templates and interim records for AtlasBadge V1.0 investigations that are not yet complete.

A file in this directory:

- must state `Evidence status: In progress`;
- must not claim a Passed result without verified evidence;
- must not contain invented dates, builds, measurements or tool results;
- must keep unknown values explicit;
- must separate planned work from completed execution;
- must not be treated as a final Test Lead decision;
- must be moved or reclassified only after the investigation and required validation are complete.

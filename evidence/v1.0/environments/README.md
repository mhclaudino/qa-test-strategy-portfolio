# Environment Evidence

This directory contains selected evidence of the environments and configurations used to validate AtlasBadge V1.0.

Evidence may include:

* Windows desktop configuration;
* Microsoft Edge version;
* Google Chrome desktop version;
* Samsung Galaxy S20 FE and Android Chrome version;
* Redmi Pad 2 evidence when additional tablet coverage is performed;
* desktop resolution;
* portrait and landscape orientation checks;
* browser and device limitations;
* Firebase project verification;
* Firebase Emulator configuration;
* confirmation of safe Playwright execution conditions.

Each evidence item should identify:

* date of execution;
* environment;
* browser or device;
* relevant version;
* purpose of the verification;
* result;
* limitation or follow-up action when applicable.

Do not include passwords, API keys, environment files, tokens or sensitive configuration values.
